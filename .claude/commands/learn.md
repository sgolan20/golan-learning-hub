# Learning Hub Assistant - SKILL

## תיאור
skill זה מיועד לסייע בלמידה ופיתוח. לאחר כל תשובה על שאלה, יש לבצע commit אוטומטי אך **לא** לעשות push ל-GitHub.

## כללי התנהגות

### 1. מענה על שאלות
- ענה על השאלה בצורה מפורטת וברורה
- הוסף דוגמאות קוד כאשר רלוונטי
- הסבר את הלוגיקה מאחורי התשובה

### 2. יצירת/עדכון קבצים
- אם התשובה דורשת יצירת או עדכון קבצים - בצע זאת
- שמור על מבנה הפרויקט הקיים
- הוסף הערות בעברית בקוד כאשר רלוונטי

### 3. ביצוע Commit אוטומטי
לאחר כל מענה על שאלה שכלל שינויים בקבצים:
1. הוסף את כל הקבצים שהשתנו ל-staging: `git add .`
2. בצע commit עם הודעה מתאימה בעברית או אנגלית
3. **חשוב: לא לבצע git push!**

### פורמט הודעת Commit
```
[Learning] <נושא התשובה>

<תיאור קצר של מה נוסף/שונה>

🤖 Generated with Claude Code
```

### 4. מה לא לעשות
- ❌ לא לעשות `git push`
- ❌ לא לפרסם ל-GitHub
- ❌ לא לשנות את ה-remote

## דוגמה לזרימת עבודה

שאלה: "איך עובד forEach ב-JavaScript?"

1. מענה על השאלה עם הסבר ודוגמאות
2. יצירת/עדכון קובץ דוגמה אם נדרש
3. ביצוע:
   ```bash
   git add .
   git commit -m "[Learning] forEach in JavaScript - הוספת הסבר ודוגמאות"
   ```
4. **לא מבצעים push!**
