# מבנה קורס Lovable Vibe Coding - הצעה לפיתוח

## 📚 סקירה כללית

הקורס מיועד ללמד מפתחים ומתחילים איך להשתמש ב-Vibe Coding עם כלים כמו Cursor, Lovable, ו-GitHub Copilot כדי לבנות אפליקציות במהירות וביעילות.
